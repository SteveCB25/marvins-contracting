# Marvin's Contracting Portfolio Website

## 🏗️ Professional Contractor Portfolio

A modern, bilingual portfolio website for Marvin's Contracting featuring:

- ✅ **76 Real Project Photos** - Categorized gallery (Kitchen, Bathroom, Flooring, Painting)
- ✅ **Bilingual Support** - English & Spanish toggle
- ✅ **Mobile Optimized** - Swipeable carousel on mobile
- ✅ **Contact Form** - Saves inquiries to database
- ✅ **Professional Design** - Deep charcoal, white, and gold aesthetic
- ✅ **WhatsApp Ready** - Notification system for new inquiries

## 🚀 Tech Stack

**Frontend:**
- React 19
- Tailwind CSS
- Shadcn/ui components
- Lucide React icons

**Backend:**
- FastAPI (Python)
- MongoDB
- Pydantic for validation

**Hosting:**
- Frontend: Vercel
- Backend: Railway
- Database: MongoDB Atlas
- Domain: marvinscontracting.com

## 📦 Project Structure
